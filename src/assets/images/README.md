# Assets Directory

This directory should contain the following images for the app to work properly:

## Required Images

1. **icon.png** (1024x1024 px)
   - App icon for both iOS and Android
   - Should be a square image with transparent background recommended

2. **splash.png** (1242x2436 px - iPhone X/11 Pro Max resolution)
   - Splash screen shown when app is loading
   - Can be resized automatically for other devices

3. **adaptive-icon.png** (1024x1024 px)
   - Android adaptive icon
   - Should have safe area in the center (672x672 px)

4. **favicon.png** (48x48 px)
   - Web app favicon
   - Will be shown in browser tabs

5. **notification-icon.png** (96x96 px)
   - Icon shown in push notifications
   - Should work well at small sizes

## Temporary Placeholder

For development purposes, you can use solid color squares or the Foundation EEP logo.
The app will still run without these images, but may show warnings.

## Production

Before deploying to production, make sure all images are:
- High quality
- Properly sized
- Follow the brand guidelines
- Optimized for file size
